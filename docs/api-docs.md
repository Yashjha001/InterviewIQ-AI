# API Docs

## Base URL

`/api`

## Auth

- `POST /auth/register`
- `POST /auth/login`
- `GET /auth/health`

## Resume

- `POST /resume/upload`
- `GET /resume/health`

## Interview

- `POST /interview/start`
- `POST /interview/submit`
- `GET /interview/health`

## Feedback

- `POST /feedback/generate`
- `GET /feedback/health`

## Report

- `GET /report/:id`
- `GET /report/health`

## Notes

This file is a starter reference for the backend API surface and can be expanded with request and response examples as endpoints are finalized.
